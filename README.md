## Step One: Get socialfocus
### Firefox (Zen, Librewolf)
1. [Install](https://addons.mozilla.org/firefox/downloads/file/4567333/socialfocus-7.2.xpi)
### Chrome (Brave, Vivaldi, Chromium, Opera, OperaGX, Yandex, Samsung Internet)
1. [socialfocus storepage](https://chromewebstore.google.com/detail/socialfocus-hide-distract/abocjojdmemdpiffeadpdnicnlhcndcg)
2. Click 'add extension'


## Step Two: Choose a config
- [Remove recommended engines](https://raw.githubusercontent.com/toastxc/social-focus-templates/refs/heads/main/configs/remove-recommended-engine.txt)

## Step Two Install it
3. Control A + Control C
4. Open Socialfocus -> Hamburger Menu -> Import / Export -> Paste Exported Settings -> Import

## All done!
